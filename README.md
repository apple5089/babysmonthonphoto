# 照片自动添加时间戳程序

## 功能说明
1. 自动读取照片文件名中的时间信息
2. 将时间添加到照片右下角，格式如 `2022.09.21`
3. 使用 Times New Roman 字体
4. 保持原图画质和比例不压缩

## 安装依赖

```bash
pip install -r requirements.txt
```

或直接安装：
```bash
pip install Pillow
```

## 使用方法

### 1. 准备照片
将需要处理的照片放到程序所在目录

### 2. 文件命名格式
程序支持以下日期格式的文件名：
- `照片2022.09.21.jpg`
- `IMG_2022-09-21.png`
- `20220921_photo.jpg`
- `2022_09_21.bmp`

### 3. 运行程序
双击运行 `add_timestamp_to_photos.py` 或在命令行执行：
```bash
python add_timestamp_to_photos.py
```

### 4. 查看结果
处理后的照片会保存在程序目录下的 `output` 文件夹中

## 程序特点
- 一键批量处理多张照片
- 自动
- 从文件名提取日期
- 字体大小根据图片尺寸自动调整
- 白色文字配黑色阴影，确保在各种背景上清晰可见
- 保持原图画质和尺寸

## 注意事项
- 确保 Windows 系统已安装 Times New Roman 字体（系统自带）
- 如果找不到 Times New Roman 字体，将使用默认字体
- 建议先备份原照片
